# Security Policy

The WINDI project is designed around cryptographic integrity and privacy-preserving verification.

## Reporting a Vulnerability

Please report security issues privately via:

**security@windi.systems**

Do NOT open public issues for security vulnerabilities.

## Scope

Security considerations include:

- Verification logic correctness
- Hash canonicalization integrity
- Signature validation correctness
- Policy decision determinism

## Response Timeline

- Acknowledgment within 48 hours
- Initial assessment within 7 days
- Resolution timeline communicated after assessment
