Grundlegendes Prinzip Foundational Principle Princípio Fundamental

Kern-Governance-Prinzip Core Governance Principle Princípio Central de Governança

Das WINDI System besitzt keine Entscheidungsbefugnis. The WINDI System does not possess decision-making authority. O Sistema WINDI não possui autoridade de tomada de decisão.

Alle institutionelle Verantwortung verbleibt bei menschlichen Operatoren und Governance-Rollen. All institutional responsibility remains with human operators and governing roles. Toda responsabilidade institucional permanece com operadores humanos e funções de governança.

Systemausgaben sind beratend und unterstützend, niemals autonom oder selbstautorisierend. System outputs are advisory and assistive, never autonomous or self-authorizing. Saídas do sistema são consultivas e assistivas, nunca autônomas ou autoautorizadas.

Human-in-the-Loop Anforderung Human-in-the-Loop Requirement Requisito Human-in-the-Loop

Jeder operative Zyklus muss direkte menschliche Beteiligung beinhalten: Every operational cycle must include direct human involvement: Cada ciclo operacional deve incluir envolvimento humano direto:

  1. Ein Mensch initiiert eine Aufgabe A human initiates a task Um humano inicia uma tarefa
  2. Das System generiert eine Antwort The system generates a response O sistema gera uma resposta
  3. Ein Mensch überprüft die Ausgabe A human reviews the output Um humano revisa a saída
  4. Ein Mensch genehmigt, bearbeitet oder lehnt explizit ab A human explicitly approves, edits, or rejects Um humano aprova, edita ou rejeita explicitamente

Kritische Regel Critical Rule Regra Crítica

Keine Systemausgabe darf in einem institutionellen Kontext ohne menschliche Validierung verwendet werden. No system output may be used in an institutional context without human validation. Nenhuma saída do sistema pode ser usada em contexto institucional sem validação humana.

Aufsichtsrollen Oversight Roles Funções de Supervisão

Operatoren Operators Operadores

Verantwortlich für das Einreichen von Aufgaben, die Überprüfung von Ausgaben und die Sicherstellung der Angemessenheit. Erste Ebene der Aufsicht. Responsible for submitting tasks, reviewing outputs, ensuring appropriateness. First level of oversight. Responsáveis por submeter tarefas, revisar saídas, garantir adequação. Primeiro nível de supervisão.

Guardian Guardian Guardian

Bietet institutionelle Aufsicht. Definiert akzeptable Nutzungsgrenzen, priorisiert Schutzmaßnahmen, interveniert bei Bedarf. Governance-Autorität über Systemverhalten. Provides institutional oversight. Defines acceptable use boundaries, prioritizes safeguards, intervenes when needed. Governance authority over system behavior. Fornece supervisão institucional. Define limites de uso aceitável, prioriza salvaguardas, intervém quando necessário. Autoridade de governança sobre comportamento do sistema.

Architect Architect Architect

Verantwortlich für Systemdesign und strukturelle Integrität. Stellt sicher, dass technische Fähigkeiten niemals menschliche Autorität umgehen. Responsible for system design and structural integrity. Ensures technical capabilities never bypass human authority. Responsável pelo design do sistema e integridade estrutural. Garante que capacidades técnicas nunca contornem autoridade humana.

Interventionsrechte Intervention Rights Direitos de Intervenção

Menschliche Aufsichtsrollen haben die Befugnis zu: Human oversight roles have the authority to: Funções de supervisão humana têm autoridade para:

  • Die Systemnutzung jederzeit stoppen Halt system use at any time Interromper uso do sistema a qualquer momento
  • Systemausgaben überschreiben Override system outputs Sobrescrever saídas do sistema
  • Spezifische Anwendungsfälle einschränken oder aussetzen Restrict or suspend specific use cases Restringir ou suspender casos de uso específicos

Grundlegende Regel Fundamental Rule Regra Fundamental

Der Systembetrieb ist von fortgesetzter menschlicher Genehmigung abhängig. System operation is conditional upon continued human approval. A operação do sistema é condicional à aprovação humana contínua.
